IMPORTANT : Outil manquant

Pour que la fonctionnalité "IA Scan" puisse lire les PDF, vous devez télécharger l'outil "Xpdf tools" (qui contient pdftotext.exe).

1. Allez sur : https://www.xpdfreader.com/download.html
2. Téléchargez la version Windows (zip).
3. Dans le zip, cherchez le fichier `bin64/pdftotext.exe` (ou `bin32` si vous êtes en 32-bit).
4. Copiez ce fichier `pdftotext.exe` ICI, dans ce dossier `tools`.

Sans ce fichier, l'extraction de texte ne fonctionnera pas.
